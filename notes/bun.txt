# What is bun?

Like Node.js, Bun is a JavaScript runtime that allows you to execute JavaScript code on the server side.

## Installing bun

Linux

```jsx
curl -fsSL https://bun.sh/install | bash
```

Mac

```jsx
powershell -c "irm bun.sh/install.ps1 | iex"
```
